JSE Module 6 Unit 2 **workshop** lab: implement full CRUD, correct HTTP status codes, and optional best practices (env config, `@Transactional`, JDBC) **after** the Unit 2 workshop.

## Lab instructions

See **[skillable-lab-w2-spring-boot-api.md](./skillable-lab-w2-spring-boot-api.md)** for full task descriptions, setup, and success criteria.

## When to use

Complete this lab in the **two hours after** the Unit 2 workshop has ended.

- [start.spring.io](https://start.spring.io/)

- [Hoppscotch](https://hoppscotch.io/) (Browser Extension for localhost) or Postman
